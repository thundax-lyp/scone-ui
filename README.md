# scone-ui

React and TailwindCSS frontend project.

## Runtime

- Node.js 22+
- pnpm 10+

## Development

```sh
pnpm install
pnpm run dev
pnpm run format:check
pnpm run lint
pnpm run typecheck
pnpm run test
pnpm run build
```

## Governance

- Documentation router: `docs/AGENTS.md`
