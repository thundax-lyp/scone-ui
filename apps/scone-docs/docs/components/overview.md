# 组件总览

Scone UI 按后台界面常见任务组织组件：

| 分类 | 组件示例 | 用途 |
| --- | --- | --- |
| 表单 | `SconeForm`、`SconeField`、`SconeInput` | 收集、校验和提交结构化信息。 |
| 数据展示 | `SconeTable`、`SconeDescriptions`、`SconeList` | 展示记录、详情和列表信息。 |
| 导航 | `SconeMenu`、`SconeTabs`、`SconeBreadcrumb` | 组织页面入口和局部视图切换。 |
| 反馈 | `SconeAlert`、`SconeToast`、`SconeProgress` | 呈现状态、提示和进度。 |

组件文档应优先说明公共 API、可访问性要求、组合方式和边界条件。
