# Admin Patterns Spec

本文档是 Admin Pattern 和 Pattern Recipe 的索引页。具体合同已拆分到 `patterns/` 和 `recipes/`。

## Purpose

本文档定义高频 Admin UI 组合模式。Pattern 提供推荐结构和组合边界，但默认不形成单个万能组件 API。
