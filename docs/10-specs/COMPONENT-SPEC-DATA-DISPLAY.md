# Data Display Component Spec

本文档是该分组的索引和共享规则。单组件合同已拆分到更小颗粒度的 SPEC 文件。

## Scope

本规格覆盖 admin-ui 基础数据展示组件。组件库只抽象业务中性的展示、状态和组合边界，不绑定业务数据模型、请求协议、权限判断或路由。

跨组件状态、尺寸和词汇以 [`FOUNDATIONS-SPEC.md`](./FOUNDATIONS-SPEC.md) 为准；DataTable 组合以 [`ADMIN-PATTERNS-SPEC.md`](./ADMIN-PATTERNS-SPEC.md) 为准。
